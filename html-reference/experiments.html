HTML Experiments: Break Things, Push Edges, Get Weird

This page contains three deliberately strange HTML demos. They use native HTML in unusual ways, stretch semantics, abuse widgets, and explore how weird a page can get while still being mostly real HTML.

Experiments on this page
  • Experiment 1: Disclosure Labyrinth
  • Experiment 2: Faux OS Dashboard

Experiment 1

Disclosure Labyrinth

This experiment abuses <details> and <summary> by turning them into a nested universe, treasure hunt, and collapsible narrative structure.

Open the universe
Enter the galaxy
Approach the solar system
Land on the planet
Walk into the city
Find the suspicious building
Find the suspicious door
Find the suspicious room
Find the suspicious fridge
Open the fridge of destiny

You found: one glowing grape, two existential breadcrumbs, and a receipt from 1997.

This is weird because disclosure widgets are being used as spatial navigation instead of simple FAQ toggles.

Why this is weird

  • It turns native disclosure widgets into level design.
  • It uses no framework and almost no JavaScript.
  • It feels interactive even though it is mostly just nested HTML.

Edge pushed

This takes an accessibility-friendly element and repurposes it into a nested narrative machine.
